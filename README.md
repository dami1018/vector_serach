# vector_serach
在大模型应用中，词向量检索即RAG的方法在应用层面是个黑盒，应用者能够做的就是选取合适的向量数据库以及向量化大模型，对背后的机制了解的很少，这样带来的弊端是如果RAG结果不好，不知道如何去改进，所以在这里，新增了词向量搜索的新专题，对词向量进行进一步研究，现在研究的是基于langchain的词向量库chroma
